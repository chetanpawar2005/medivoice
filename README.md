# 🏥 MediVoice - Smart Healthcare Assistant

## 📌 Overview

MediVoice is a smart healthcare web application that helps users find **nearby doctors, clinics, and hospitals** based on their symptoms. It also provides **navigation support using maps** to reach the selected hospital or clinic easily.

---

## 🚀 Features

* 🔍 Find nearby doctors and clinics
* 🏥 Locate nearby hospitals
* 🗺️ Get directions using map integration
* 🧠 Symptom-based doctor recommendations
* 📍 Real-time location-based results
* 💬 Simple and user-friendly interface
